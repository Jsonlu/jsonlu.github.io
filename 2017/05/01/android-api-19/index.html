android-api-19

作者 JsonLu 日期 2017-05-01
android-api-19

变更

1.读取应用外部共享文件（getExternalStoragePublicDirectory()）需要 READ_EXTERNAL_STORAGE权限，访问getExternalFilesDir()不需要。
2.WebView底层支持升级为Chromium源代码快照版本，支持H5远程调试
3.沉浸式全屏模式
4.透明系统状态栏，使用新主题背景 Theme.Holo.NoActionBar.TranslucentDecor 和 Theme.Holo.Light.NoActionBar.TranslucentDecor 将系统状态栏设置为部分透明。
