macaca自动化测试（安卓）

介绍

Macaca 是一套面向用户端软件的测试解决方案，提供了自动化驱动，周边工具，集成方案，旨在解决终端上的测试、自动化、性能等方面的问题。
本示例采用nodejs方案，关于node如何安装请自行解决。

//安装工具包和驱动
$ npm i macaca-cli macaca-ios -g

//安装uirecorder
$ npm install uirecorder mocha -g

//创建测试并初始化项目
$ mkdir test && cd test
$ uirecorder init && npm i

//启动服务
$ macaca server –port 4444 –verbose
//开始移动端录制,输入文件名称，开始录制，结束录制
$ uirecorder start –mobile

//Macaca 运行（普通运行）
$ macaca run -p 4444 -d test.js –verbose
//Macaca 运行（会生成报告）
$ source run.sh sample/test.spec.js (Linux|Mac) or run.bat sample/test.spec.js(Windows)

报告在reports/目录下
