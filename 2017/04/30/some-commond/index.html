RSA密钥生成命令

//生成RSA私钥
openssl>genrsa -out rsa_private_key.pem 2048
//生成RSA公钥
openssl>rsa -in rsa_private_key.pem -pubout -out rsa_public_key.pem
//将RSA私钥转换成PKCS8格式
openssl>pkcs8 -topk8 -inform PEM -in rsa_private_key.pem -outform PEM -nocrypt
注意：“>”符号后面的才是需要输入的命令。

版本规则

主版本号.次版本号.修订号
主版本号：做了不兼容的 API 修改
次版本号：做了向下兼容的功能性新增
修订号：做了向下兼容的 bug 修正

keytool证书操作

//生成证书
keytool -genkey -alias test -keyalg RSA -keysize 1024 -keystore debug.jks -validity 365000
//获取证书指纹
keytool -list -v -keystore /Users/jsonlu/WorkSpace/android/latteread3/conf/debug.jks |grep MD5
//获取app签名指纹
keytool -printcert -file /Users/jsonlu/Desktop/CERT.RSA
//获取证书库信息
keytool -list -rfc -keystore debug.jks
//jarsigner签名
jarsigner -keystore debug.jks /Users/jsonlu/WorkSpace/idea/wechat/build/libs/wechat-service-0.1.0.zip test
